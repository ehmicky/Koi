Structure:
  - start byte:
     - single sequence
     - either:
        - \x1b[
           - i.e. \e[
        - \x9b (CSI, Control Sequence Introducer)
  - parameter bytes:
     - 0-n characters
     - among \x30-\x3f
        - i.e. 0-9 : ; < > = ?
     - often use ;-separated digits
        - with omitted digits being 0
  - intermediate bytes:
     - 0-n characters
     - among \x20-\x2f
        - i.e. space ! " # $ % & ' ( ) * + , - . /
  - final byte:
     - single character
     - among \x40-\x7e
     - A–Za-z @ [ ] { } \ ^ _ ` | ~
     - often use m
